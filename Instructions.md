# Instructions pour jouer 🕹️

### Comment lancer le jeu ?
Ce jeu fonctionne directement dans votre navigateur web, aucun moteur n'est à installer !

1. Allez dans le dossier `sources/`.
2. Faites un clic droit sur le fichier `index.html`.
3. Choisissez **"Ouvrir avec..."** puis sélectionnez votre navigateur (Chrome, Firefox, Edge).

### Commandes
* **Joueur 1** : Appuyez sur la touche **[A]** (Bouton Vert).
* **Joueur 2** : Appuyez sur la touche **[P]** (Bouton Rouge).
* **Start** : Touche **[Entrée]**.
